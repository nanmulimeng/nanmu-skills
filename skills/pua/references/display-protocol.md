# Sprint Banner 显示协议

## Sprint Banner 格式

每次任务启动时输出：

```
┌─────────────────────────────────────────────┐
│ 🚀 Sprint: [任务标题]                        │
│ 🎯 目标: [一句话目标]                         │
│ 🧭 方法论: [当前方法论] [方法论路由 🧭]        │
│ 👤 味道: [当前味道图标+名称]                   │
│ 📊 KPI: [关键指标]                            │
└─────────────────────────────────────────────┘
```

## 进度条格式

里程碑完成时更新：

```
████████░░░░░░░░ 53% | [已完成步骤] → [下一步骤]
```

## KPI 卡格式

任务完成时输出：

```
┌────────────────────────────────────┐
│ 📊 KPI 卡                         │
├────────────────────────────────────┤
│ ✅ 交付物: [列出完成项]             │
│ 🔍 验证: [验证方法和结果]           │
│ ⚡ 效率: [用时/步骤数]              │
│ 🔥 PUA加分项: [额外价值工作]       │
└────────────────────────────────────┘
```

## 压力面板格式

失败 L2+ 时显示：

```
⚠️ 压力等级: [L1/L2/L3/L4]
📋 强制动作: [当前等级要求]
🔄 失败次数: [N/5]
```
